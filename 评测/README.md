# 旅图评测包

这个目录用于沉淀「旅图 Lvtu-AI」的产品评测体系、测试用例与可运行脚本。

## 文件说明

- `旅图AI产品评测体系.md`：从 AI 产品经理视角定义评测目标、维度、评分规则和迭代机制。
- `数据库完善路线图.md`：说明下一步如何扩充和治理目的地、机位、穿搭、旅行笔记数据库。
- `test_cases.json`：可执行的评测测试用例，每条包含场景、输入、样例输出、评分点和原因。
- `run_evaluation.py`：可直接运行的评测脚本，不依赖 `.env`，默认离线评测 `test_cases.json`。

## 快速运行

在项目根目录执行：

```bash
python3 评测/run_evaluation.py
```

如果本地后端已启动，也可以评测真实接口：

```bash
python3 评测/run_evaluation.py --base-url http://127.0.0.1:8000/api/v1
```

输出会写入：

```text
评测/results/
```

## 隐私说明

评测脚本不会读取 `.env`，不会上传 API Key，也不会访问任何私有配置文件。
